# 💼 SkyLine Venture Expenses Tracker

A comprehensive Excel-based expense tracking and analysis system for multi-company ventures with interactive dashboards, vendor tracking, and quarterly/annual reporting capabilities.

## 📋 What This Does

**Solves the problem:** How to track and analyze operating expenses across 20-30 companies, identify spending patterns, compare vendors, and generate financial reports by quarter, year, and financial year.

**Key Benefits:**
- Track all expenses for multiple venture companies
- Analyze spending patterns by company, vendor, and expense type
- Generate quarterly and annual reports automatically
- Compare expenses across companies and years
- Monitor vendor billing and consumption costs
- Complete audit trail from 2022-2025

## 🚀 How to Use

### Quick Start (First Time Setup)

**Step 1: Enable Macros (IMPORTANT!)**
1. Download file from link above
2. Open in Microsoft Excel
3. **Allow Macros** when prompted:
   - Click "Enable Macros" or
   - Click "Enable Content" or
   - Check "Enable editing" → "Enable macros"
4. Wait for dashboard to load

**Step 2: Use Interactive Dashboard**
1. Go to **"DashBoard"** sheet
2. Find the dropdown filters (Company selector visible)
3. **Select Company**: Choose from list
4. View updated charts and metrics automatically
5. Try other companies to compare

**Step 3: View Reports**
1. **"Comparison"** sheet: Visual comparisons
2. **"Table"** sheet: Financial year table view
3. **"Comparison_Table"** sheet: Company & year comparisons
4. **"Dasboard_Table"** sheet: Year-wise and quarter-wise pivot tables

### Using the Dashboard

**Company Filter:**
- Click dropdown showing company names
- Select any company (BluePeak Ventures, Nexora Digital Labs, etc.)
- Dashboard automatically updates
- Charts show that company's data

**Compare Multiple Companies:**
- Use Comparison sheet
- Side-by-side analysis
- Identify best/worst performers
- Vendor comparison available

**Financial Analysis:**
- Open "Table" sheet for India FY view
- Filter by year (2022, 2023, 2024, 2025)
- Filter by month (April-March)
- See company, expense type, vendor breakdown

## 🎓 Training & Support

### Enable Macros Troubleshooting:
**Q: "Macro not working" or "Disabled content"**
- Click yellow bar → "Enable editing" → "Enable content"
- Or: File → Info → "Enable all content"
- Or: File → Options → Trust Center → Macro Settings → Enable

**Q: Dashboard not updating?**
- Check if macros are enabled
- Click any company dropdown to trigger refresh
- Or use ribbon button to refresh manually

**Q: Slicers not visible?**
- Go to "Silcers" sheet
- Slicers are configured there
- Click dashboard slicers to filter
